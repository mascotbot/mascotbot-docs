## Troubleshooting

### Common Issues

1. **Mascot not animating**

   - Ensure the Rive component is properly integrated
   - Check if the mascot file is loaded correctly

2. **Speech not working**

   - Ensure audio permissions are granted

3. **Lip-sync issues**
   - Ensure the viseme data matches your mascot's animation
